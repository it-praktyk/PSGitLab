# Remove-GitLabRunner

## SYNOPSIS
Removes a runner.

## SYNTAX

```
Remove-GitLabRunner -Id <int>
```

## DESCRIPTION
Removes a runner from GitLab.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Remove-GitLabRunner -Id 400
```

## PARAMETERS

### -Id
The ID of the GitLab runner to remove.
